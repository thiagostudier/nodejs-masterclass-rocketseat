## Introdução

### O que é?

- JS Runtime Enviroment, ambiente de execução de JS
- Não é um framework
- Não é uma linguagem

### Para quê serve?

- Backend
- Frontend
- Micro serviços
- Scripts e Automações
- API
- - Web
- - Mobile
- - Desktop

### Vantagens?

- Rápido
- - Execução
- - Prototipar aplicações
- Alta escalabilidade (aumentar volume de atividades sem aumentar o custo da aplicação)
- Aplicações de ponta
- - Netflix
- - Paypal
- Ecosistema Gigante (NPM)
- JS Everywhere
